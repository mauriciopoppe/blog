# Tech Stack

## Core Technologies
- **Hugo:** The primary static site generator used for its speed and flexible content organization.
- **Bun:** Used as the JavaScript runtime and package manager for efficient dependency management and task execution.
- **Webpack:** Manages the bundling of complex interactive scripts and React/Three.js/d3 components.

## Frontend
- **React:** Powers interactive components and complex UI logic in the sandbox and specialized pages.
- **TypeScript:** Provides type safety and better developer experience across the JavaScript codebase.
- **Tailwind CSS:** Used for rapid, utility-first styling with support for multiple themes (light/dark).
- **Three.js & d3.js:** Core libraries for creating high-performance 3D graphics and data-driven visualizations in the sandbox.

## Infrastructure & Tooling
- **Netlify:** Handles deployment, hosting, and continuous integration.
- **PostCSS:** Processes Tailwind CSS and other styling optimizations.
- **ESLint & Prettier:** Ensures code quality and consistent formatting.
